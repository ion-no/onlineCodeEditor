#ChatsApp

The project implements a online code editor compartible with C, C++, Python

Deployed @

## Getting Started

for localhost

clone the repo

### Prerequisites

install node packages

```
npm install 
```

### Installing

to run the server side


```
npm start
```

to run client

```
cd client
npm run react
```
