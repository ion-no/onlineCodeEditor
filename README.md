#Online Compiler

The project implements a online code editor compartible with C, C++, Python

Deployed @ https://onlineide-135.herokuapp.com/

## Getting Started

for localhost

clone the repo

### Prerequisites

install node packages

```
npm install
```

### Installing

to run the server side

```
npm run server
```

to run client

```
npm run react
```
